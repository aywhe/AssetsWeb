# 简单的网站

通过网页形式在局域网中分享电脑中的资源，包括图片，音乐，视频。适合家庭网，可以在手机上看电脑硬盘上的电影。  

# 依赖

需要安装 nodejs, npm 和 modules。

## modules

* "express": "^5.1.0"
* "readdir": "^1.0.2"
* "ftp-srv": "^2.16.2",
* "get-intrinsic": "^1.3.0",
* "node-cache": "^5.1.2",
* "winston": "^3.17.0"

# 配置文件

编辑 [assets_config.json](assets_config.json) 文件，并保存到项目根目录中。

# 运行

命令行中运行：

    node server.js
